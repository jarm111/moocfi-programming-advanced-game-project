# MOOC.fi Programming Advanced Game Project

Game project for University of Helsinki's MOOC Advanced Course in Programming 2026. Powered by Python and Pygame.

## Getting started

- Requires UV
- clone the repo
- `uv sync` to install dependencies
- Run `main.py`

## Gameplay and features

- Collect coins and proceed to next level while evading monsters
- Player controlled character follows mouse cursor
- Coins and monsters have random spawn locations
- Level progression adds more coins and monsters

## Project requirements and limitations

- Use only provided assets
- All the code needs to be in single file
- The game has a sprite the player can move in some way
- The game has some collectable items and/or enemies
- The player needs to be set a clear task in the game
- The game contains a counter which tells the player how they are doing in the game
- The source code for the game is divided into functions
